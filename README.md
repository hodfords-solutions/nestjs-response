# nestjs-response
For response, use sync, not async.

## Installation ðŸ¤–

```
npm install @hodfords/nestjs-response --save

```

```typescript
class UserResponse {
    @ApiProperty()
    @IsNotEmpty()
    @IsString()
    name: string
}

class UserController{
    @Get()
    @ResponseModel(UserResponse, true)
    getAllUser() {
        return [{name: "hello"}]
    }

    @Get()
    @ResponseModel(UserResponse, true, true)
    getUser() {
        return undefined
    }
}

```
